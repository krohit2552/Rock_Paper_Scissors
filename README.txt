# Rock-Paper-Scissors Game

A simple Rock-Paper-Scissors game built with HTML, CSS, and JavaScript. The game allows you to play against the computer, keeping track of scores and providing feedback on the outcome of each round.

## Features

- **Interactive Gameplay**: Play Rock, Paper, Scissors against the computer.
- **Score Tracking**: The game keeps track of both the user and computer scores.
- **Sound Effects**: Enjoy click sound effects when making a choice.
- **Responsive Design**: The game is designed to work well on various screen sizes.

## Demo

You can play the game by opening the `index.html` file in your browser.

Usage
Select one of the three options: Rock, Paper, or Scissors.
The computer will randomly select its choice.
The game will display the outcome of the round and update the scores accordingly.
Continue playing rounds until you want to reset the game.
Files
index.html: The main HTML file that sets up the structure of the game.
style.css: The CSS file that styles the game.
app.js: The JavaScript file containing the game logic.
click_sound.mp3: The sound file that plays when you make a selection.
Customization
You can customize the game by:

Changing the sound effect by replacing click_sound.mp3 with your own audio file.
Modifying the styles in style.css to fit your design preferences.
Adding new features or game modes in the app.js file.
